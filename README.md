# itcp
Advanced Network Pinger with various options.

# Why use itcp?
* TCP and ICMP support
* Hostname Lookup
* Threads
* Timeout
* ISP lookup

# Download
* Clone repo
* When you are done with downloading just drag the exe into C:\Windows\System32
* Final Step: Just open command prompt and type itcp
